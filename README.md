# Sildo

**Slide in. Do more with prompts.**

A Chrome extension that adds a convenient sidebar to ChatGPT, allowing you to quickly navigate through your conversation prompts.

*Sildo = Slide + Do*

## Features

- **Sidebar Navigation**: A collapsible sidebar that displays all your prompts in the current conversation
- **Quick Jump**: Click on any prompt in the sidebar to instantly scroll to it in the conversation
- **Visual Highlight**: Selected prompts pulse briefly for easy identification
- **Toggle Button**: Arrow button to show/hide the sidebar
- **Clear Function**: Clear button to remove all prompts from the sidebar view
- **Auto-Update**: Automatically detects and adds new prompts as you chat

## Installation

1. Clone or download this repository to your local machine
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the folder containing this extension
5. The extension is now installed and ready to use

## Usage

1. Navigate to [ChatGPT](https://chat.openai.com)
2. Start or open a conversation
3. Click the arrow button on the right side of the screen to open the sidebar
4. Your prompts will appear numbered in the sidebar
5. Click any prompt to jump to it in the conversation
6. Click "Clear" to empty the sidebar view
7. Click the arrow button again to hide the sidebar

## Files

- `manifest.json` - Extension configuration
- `content.js` - Main functionality and UI logic
- `styles.css` - Styling for the sidebar and UI elements
- `icons/` - Extension icons

## Requirements

- Google Chrome browser
- Access to ChatGPT (chat.openai.com)
